# Workflow: Partner

> Übersicht: [README](../../README.md) · Rollen: [docs/roles-and-permissions.md](../roles-and-permissions.md)

## Ziel

Sichere Referral-Erfassung und transparente Partnerkommunikation ohne Überschreitung von Berechtigungsgrenzen.

## Ablauf

1. Partner-Login
2. Referral-Erfassung
3. Status-Verfolgung der eigenen Referrals
4. Partner-Kommunikation / Einstellungen

## Kritische UX-Punkte

- Nur partner-eigene Datensicht.
- Eindeutige Übergabepunkte an Staff.
